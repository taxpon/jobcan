Jobcan
======

Script for jobcan automation.

Install
-------
.. code-block:: bash

    pip install jobcan


Usage
-----

.. code-block:: bash

    Usage: jobcan [OPTIONS]

    Options:
      -c, --command TEXT
      -e, --email TEXT
      -p, --password TEXT
      -g, --gid INTEGER
      -c, --cid TEXT
      -v, --version
      --help               Show this message and exit.
